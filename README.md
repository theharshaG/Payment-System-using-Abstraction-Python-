# Payment System using Abstraction (Python)

## Project Overview
This project demonstrates the concept of **Abstraction** in Python using the `abc` (Abstract Base Class) module.
A base abstract class called **Payment** defines a common method `pay()`.  
Different payment methods such as **UPI**, **Card**, and **Wallet** implement this method in their own way.
The program asks the user to select a payment method and then processes the payment accordingly.

## Features

- Demonstrates **Abstraction using Abstract Base Classes**
- Multiple payment methods (UPI, Card, Wallet)
- Wallet balance management
- Simple menu-driven interface
- Real-world payment system simulation

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- Abstract Base Classes (`abc` module)
